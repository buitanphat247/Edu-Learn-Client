# Tạo ngân hàng câu hỏi

Xây dựng kho câu hỏi để sử dụng lâu dài cho nhiều đề thi khác nhau.

## Thêm câu hỏi thủ công
1. Vào mục **Ngân hàng câu hỏi**.
2. Chọn **Thêm câu hỏi**.
3. Chọn dạng câu hỏi (Trắc nghiệm 1 đáp án, nhiều đáp án, đúng/sai...).
4. Nhập nội dung câu hỏi và các phương án.
5. Chọn đáp án đúng.

## Nhập từ Excel (AI hỗ trợ)
Bạn có thể tải lên file Excel chứa danh sách câu hỏi. Hệ thống AI sẽ tự động đọc và phân loại câu hỏi vào ngân hàng của bạn.
